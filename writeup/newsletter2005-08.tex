\QTP{H2}
\label{55}\textbf{Version 5.5 shipping now}

\QTP{genpara}
MacKichan Software is pleased to announce Version 5.5 of \QTR{products}{%
Scientific WorkPlace}, \QTR{products}{Scientific Word}, and \QTR{products}{%
Scientific Notebook}. With the release of Version 5.5, MacKichan Software
brings a new computational engine, enhanced graphics, and extended LaTeX
typesetting capabilities to your workplace.

\QTP{genpara}
Note these new features in Version 5.5:

\begin{itemize}
\item Compute and plot using the \QTR{products}{MuPAD 3} computer algebra
engine

\item Animate 2D and 3D plots using \QTR{products}{MuPAD's} VCAM

\item Rotate, move, zoom in and out, and fly through 3D plots with new
OpenGL 3D graphics

\item Label 2D and 3D plots so that the label moves when you rotate or zoom
a plot

\item Create 3D implicit plots

\item Import LaTeX files produced by other programs

\item Use many new LaTeX packages

\item Get help from the new, extensive troubleshooting section of 
\QTR{books}{Typesetting Documents with Scientific WorkPlace and Scientific
Word, Third Edition}
\end{itemize}
